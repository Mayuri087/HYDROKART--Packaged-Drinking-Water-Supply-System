# HYDROKART – Packaged Drinking Water Supply System

## 📌 Project Description
HYDROKART is a system designed to manage and supply packaged drinking water efficiently. It helps in handling customer orders, delivery tracking, and inventory management.

## 🎯 Features
- Customer order management
- Water supply tracking
- Inventory management
- User-friendly interface

## 🛠️ Technologies Used
- Java
- NetBeans IDE
- (Add database: MySQL)

## 🚀 How to Run
1. Open project in NetBeans
2. Build the project
3. Run the application

## 📂 Project Structure
- Source code
- Database files
- Project configuration files
